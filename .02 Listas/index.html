Introducción a HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje utilizado para crear páginas web. Permite estructurar el contenido mediante etiquetas.

Extensión del archivo

Los documentos HTML deben guardarse con la extensión .html

Etiquetas básicas

html: Indica el inicio del documento HTML.
head: Contiene metadatos, título de la página, enlaces a hojas de estilo, etc.
title: Define el título que aparece en la pestaña del navegador.
body: Contiene el contenido visible de la página web.

Uso de etiquetas o "tags"

Las etiquetas HTML se escriben entre los símbolos < y >. Casi todas las etiquetas tienen una etiqueta de apertura <"etiqueta"> y una de cierre <"/etiqueta">

Encabezados

<"h1">: Título principal de la página. Debe utilizarse solo una vez por temas de SEO (posicionamiento en buscadores) y accesibilidad.
<"h2"> a <"h6">: Subtítulos jerárquicos. Pueden repetirse según secciones del contenido.

Párrafos

La etiqueta <"p"> se utiliza para escribir párrafos de texto.

Importancia del uso correcto

Un uso correcto de las etiquetas ayuda a los buscadores (como Google) y a las herramientas de asistencia (lectores de pantalla) a entender mejor el contenido de la página.
